# Contributing

## Testing

```bash
bin/phpunit
```

## Code Style

```bash
bin/cs
```

## Static Analysis

```bash
bin/sa
```

## Linting

```bash
bin/lint
```

## Setup a Test Project

Create a new bare Symfony skeleton application with specific MsgPHP bundles pre-installed.

```bash
bin/create-project
```

## Perform a Smoke Test

To quickly see if CI is likely to pass, use:

```bash
bin/smoke-test
```
